###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;
greetign = {};
console.log(greetign);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i>The result is going to be undefined because 'greetign' which I am trying to console log is not declared. as a javascript variable needs to be declared with proper variable declarations such as var, let or const. as greetign is not a variable and we are trying to console log it, it's going to return undefined.</i>

</p>
